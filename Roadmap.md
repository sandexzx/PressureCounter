Техническое задание: Android приложение для мониторинга давления и пульса
1. Общее описание
Мобильное приложение для Android, позволяющее пользователям отслеживать артериальное давление и пульс с визуализацией данных и аналитикой.

2. Основной функционал
2.1 Ввод данных
Быстрое добавление измерений:
Систолическое давление (верхнее)
Диастолическое давление (нижнее)
Пульс
Дата и время (автозаполнение текущим)
Заметки (опционально)
Самочувствие (смайлики/теги)
Валидация вводимых значений
Возможность редактирования/удаления записей
2.2 Статистика и аналитика
Пульсовое давление (разница между систолическим и диастолическим)
Средние значения за период
Минимальные/максимальные показатели
Тренды (повышение/понижение)
Графики:
Линейные графики по дням/неделям/месяцам
Цветовая индикация зон (норма/повышенное/пониженное)
2.3 Экспорт данных
Форматы: PDF, CSV, Excel
Отправка через email/мессенджеры
Печать отчетов
Резервное копирование
2.4 Дополнительные функции
Напоминания о измерениях
Цели и рекомендации
История измерений (список с фильтрами)
Темная/светлая тема
3. Дизайн
3.1 Стиль
Material Design 3
Минималистичный интерфейс
Крупные, читаемые шрифты
Плавные анимации и переходы
3.2 Цветовая схема
Основные цвета: синий/зеленый (спокойные тона)
Акценты для важных действий
Цветовая индикация показателей:
Зеленый - норма
Желтый - предупреждение
Красный - критично
3.3 Экраны
Главный экран

Кнопка быстрого добавления (FAB)
Последнее измерение (крупно)
Мини-график за неделю
Экран добавления

Числовые поля с большими кнопками
Быстрый ввод
Статистика

Вкладки по периодам
Интерактивные графики
Карточки с показателями
История

Список с группировкой по датам
Свайп для удаления
4. Технические требования
4.1 Платформа
Android 14.0+ (API 34+)
Kotlin
Архитектура: MVVM / Clean Architecture
4.2 Библиотеки
Jetpack Compose / XML layouts
Room Database (локальное хранение)
MPAndroidChart / Vico (графики)
WorkManager (напоминания)
Apache POI / iText (экспорт)
4.3 Производительность
Быстрый запуск (<2 сек)
Плавная прокрутка (60 FPS)
Минимальное потребление батареи
